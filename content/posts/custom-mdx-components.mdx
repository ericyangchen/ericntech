- **LinkWithIcon**

  <div className="flex items-center">
    Check this link out:
    <LinkWithIcon href="https://ericntech.com/">
      https://ericntech.com
    </LinkWithIcon>
  </div>

  **Color Variants**:

  - _`className`_ prop will be applied to outer div and SVG,

  - _`iconClassName`_ prop will be applied to the SVG only.

  Using _`text-pink-500`_ in _`className`_ prop style both text and SVG orange, <br/>
  to style the SVG alone, use _`text-green-500`_ in _`iconClassName`_ prop. (also remember to
  add the width and height properties to the SVG)

<LinkWithIcon href="https://ericntech.com/" className="text-pink-500">
  https://ericntech.com
</LinkWithIcon>

<CH.Code>
